# yt-dlp 사용을 도와주는 GUI 프로그램

이 프로그램은 선택한 옵션에 맞는 yt-dlp 명령어를 실행하는 프로그램입니다.  
[릴리즈 페이지](https://github.com/eoehd1ek/yt-dlp-GUI-Helper/releases)에서 다운로드 받을 수 있습니다.

# 사용법
`ffmpeg`프로그램과 `yt-dlp`프로그램이 필요합니다.  
아래 경로에서 각 파일을 다운받아 동일한 폴더에 놓아야합니다.
- [yt-dlp Download](https://github.com/yt-dlp/yt-dlp/releases/)
- [ffmpeg Download](https://ffmpeg.org/download.html)

# 변경 내역

## 2026-03-01 업데이트
1. UI 변경  
pyside6를 사용하여 화면을 단순하게 변경했습니다.

2. 프리미어 프로 편집용 다운로드 옵션 추가  
사용시 h264, acc 코덱으로 영상을 저장합니다.

3. 음원 썸네일 추가  
음원 다운로드시 영상 썸네일을 포함하여 저장합니다.

4. 플레이 리스트 ID 제거 기능 추가
플레이 리스트 영상은 주소창에 플레이리스트 ID가 포함되어 모든 영상을 받는 문제가 있습니다.  
도구-플레이리스트 ID 제거 기능을 사용하여 영상 ID만 남길 수 있습니다.

## 2023-03-20 업데이트
1. 의존 프로그램 yt-dlp로 변경.  
   2021년 youtube-dl 지원이 종료되어 동영상 다운로드가 되지 않는 문제를 해결했습니다.

2. 소스코드 리펙토링을 진행했습니다.  

## 2021-12-04 업데이트

1. 폴더 이름에 공백이 존재하면 다운로드가 정상 작동하지 않는 버그 해결  
   \\\"경로명\\\" 과 같이 따옴표로 묶어주어 해결함

## 2021-03-16 업데이트

1. 다운로드시 프로그램 멈춤 개선  
   다운로드 버튼을 눌러도 프로그램이 멈추지 않습니다.  
   여러개의 다운로드 요청을 동시에 진행할 수 있습니다.
   
2. 다운로드 완료 알림 제거
   콘솔창에 youtube-dl의 결과가 표시됩니다.  
   다운로드 실패 정보를 확인하려면 콘솔을 확인하면 됩니다.
   이에 다운로드 완료 알림이 뜨지 않습니다. 

3. 메뉴바 기능 추가
   youtube-dl과 ffmpeg의 공식 사이트로 이동하는 기능이 추가됩니다.

4. 썸네일 다운로드 기능 추가
   체크박스를 활성화 하여 영상 썸네일을 함께 다운로드 받을 수 있습니다.
